# 悟空邀请码自动填写脚本

这是一个用于 Windows 桌面端的自动化脚本，目标是：

- 自动请求悟空邀请码接口
- 自动把邀请码粘贴到当前已聚焦的输入框
- 自动点击提交按钮
- 在启动后持续运行，并在每个整点前 1 秒再次尝试抢码

脚本文件：`auto_register.py`

## 功能说明

脚本执行流程如下：

1. 启动后先请求一次邀请码并尝试填写。
2. 如果失败，进入等待。
3. 每个整点前 1 秒触发请求，拿到邀请码后自动填写并点击提交。
4. 循环执行，直到你手动停止（`Ctrl + C`）。

## 环境要求

- Python 3.9+
- Windows（脚本里使用了 `Ctrl+A`、`Ctrl+V` 和固定鼠标坐标点击）
- 已安装依赖：
  - `requests`
  - `pyautogui`
  - `pyperclip`

安装依赖：

```powershell
pip install requests pyautogui pyperclip
```

## 启动方法

在项目目录执行：

```powershell
python auto_register.py
```

脚本启动后会持续运行，终止时按：

```text
Ctrl + C
```

## 重点：输入框 focus（焦点）怎么处理

这个脚本不会自己找输入框，它依赖“当前活动焦点”来输入内容。

请按下面步骤操作：

1. 打开悟空桌面端并进入邀请码输入页面。
2. 用鼠标单击邀请码输入框，让光标出现在输入框内。
3. 不要再点击其他窗口，直接回到终端启动脚本。
4. 启动后保持悟空窗口在前台，避免焦点被切走。

如果焦点不在输入框，脚本会把内容粘贴到错误位置（或无效）。

## 重点：如何获取鼠标位置（用于提交按钮坐标）

脚本当前提交按钮是固定坐标点击：

```python
pyautogui.click(x=638, y=946)
```

不同分辨率、缩放比例、窗口位置下，这个坐标大概率需要改。

### 方法 1：临时命令行查看坐标（推荐）

在新终端运行：

```powershell
python -c "import pyautogui,time; print('把鼠标移动到提交按钮上，3秒后读取坐标'); time.sleep(3); print(pyautogui.position())"
```

运行后把鼠标停在“提交按钮”上，等待输出，例如：

```text
Point(x=712, y=884)
```

然后把 `auto_register.py` 里的点击坐标改成该值：

```python
pyautogui.click(x=712, y=884)
```

### 方法 2：实时查看坐标（可选）

```powershell
python -c "import pyautogui,time; print('按 Ctrl+C 结束'); \
while True: \
    x,y=pyautogui.position(); print(f'X={x} Y={y}', end='\r'); time.sleep(0.1)"
```

把鼠标移动到目标按钮，记下坐标后 `Ctrl + C` 退出。

## 常见问题

### 1. 运行后没填到输入框

- 通常是输入框没有 focus（焦点）。
- 重新点击输入框后再运行。
- 确保运行期间不要切换到其他窗口。

### 2. 显示已提交，但界面没点中按钮

- 通常是 `pyautogui.click(x=..., y=...)` 坐标不对。
- 重新获取鼠标坐标并修改代码。

### 3. 接口请求失败

- 检查网络连接。
- 目标接口可能暂时不可用，脚本会在下一轮继续尝试。

## 注意事项

- 请在你自己的账号和合规场景下使用。
- 自动化脚本受系统缩放、分辨率、窗口布局影响较大，首次使用建议先手动验证一次流程。
